# Minimalistic Java EE 8 Microprofile Quickstarter

Java EE microprofile project quickstart template. Clean, lean and minimalistic.

Fire up your CLI and type (replace ```PROJECT_NAME``` and ```GROUP_ID```):
```
mvn archetype:generate -DarchetypeGroupId=com.airhacks -DarchetypeArtifactId=microprofile-essentials-archetype -DarchetypeVersion=0.0.3  -Darchetype.interactive=false --batch-mode -Dversion=1.0-SNAPSHOT -DgroupId=GROUP_ID -DartifactId=PROJECT_NAME
```
to create a fully fledged Java EE 8 "kB" project. Use the most recent version (here 0.0.3).

In the IDE of your choice use the Archetype GroupID ```com.airhacks```, then the Archetype ArtifactID ```microprofile-essentials-archetype``` and also select the most recent version.
